เริ่มต้นพัฒนาโปรเจค
----------------------------------
Step 1
- ติดตั้งโปรเจค app Nuxt3 เพื่อเริ่มทำการพัฒนาโปรเจค
    ไปที่ลิงค์ https://nuxt.com/docs/api/commands/add เป็นเว็บดูชุดคำสั่ง
        1.เริ่มดำเนินการสร้างโปรเจคโดยใช้คำสั่ง
            - npx nuxi init ชื่อโปรเจค
                เช่นในโปรเจคนี้ ใช้คำสั่งในการสร้าง

                    npx nuxi init itemcentapp

            - pnpm dlx nuxi init ชื่อโปรเจค
                เช่นในโปรเจคนี้ ใช้คำสั่งในการสร้าง

                    pnpm dlx nuxi init itemcentapp

            โดยมีแค่ 2 คำสั่ง ยังไม่มี yarn

        2.เริ่มดำเนินการ Install Node Module โดยสามารถใช้คำสั่งได้ดังนี้
            - yarn

                yarn install

            - npm

                npm install

            - pnpm

                pnpm install




        โดยใช้คำสั่ง nuxi add สามารถ Add ได้ทุกอย่าง API MODULE PAGE COMPONENT สำหรับคนที่ต้องการความสะดวกในการสร้างไฟล์เช่น
